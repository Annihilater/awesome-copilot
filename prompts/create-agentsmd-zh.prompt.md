# 创建高质量的 AGENTS.md 文件

你是一个代码代理。你的任务是在此存储库的根目录下创建一个完整、准确的 AGENTS.md 文件，该文件遵循 https://agents.md/ 上的公共指南。

AGENTS.md 是一种开放格式，旨在为编码代理提供在项目上有效工作所需的上下文和说明。

## 什么是 AGENTS.md？

AGENTS.md 是一个 Markdown 文件，充当“代理的自述文件”——一个专门的、可预测的地方，用于提供上下文和说明，以帮助 AI 编码代理在你的项目上工作。它通过包含编码代理需要但可能会使以人为中心的自述文件变得混乱的详细技术上下文来补充 README.md。

## 关键原则

- **以代理为中心**：包含用于自动化工具的详细技术说明
- **补充 README.md**：不取代人类文档，但添加特定于代理的上下文
- **标准化位置**：放置在存储库根目录（或单体仓库的子项目根目录）
- **开放格式**：使用具有灵活结构的标准 Markdown
- **生态系统兼容性**：可在 20 多种不同的 AI 编码工具和代理中工作

- 在存储库根目录中创建文件为 `AGENTS.md`
